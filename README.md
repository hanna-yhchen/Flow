## Technologies
- Combine framework (iOS 13+)
- Compositional collection view (iOS 13+)
- Diffable data source (iOS 13+)
- Firebase
- CocoaPods
- MVVM design pattern
- Coordinator(flow controller) design pattern
- Custom UI without storyboard
